# 追加予定の資格キュー

週次更新時に1〜2件ずつ、一次情報を確認してから `data/exams/` に追加する。
優先度は「アフィリエイト単価 × 検索需要 × 締切事故の起きやすさ」で判断。

## 優先度: 高
- [ ] 社会保険労務士(2027年度) — 2026年度試験は8/23実施・申込終了。2027年度の公示(例年4月)を待って追加
- [ ] 司法書士(2027年度) — 講座単価が高くアフィリエイト価値大。2026年度は申込終了
- [ ] 中小企業診断士(1次・2次) — 講座単価高。1次は例年8月実施・申込4〜5月
- [ ] FP1級 実技(日本FP協会・9月実施) — 申込期間の一次確認が取れ次第追加
- [ ] 保育士試験(後期) — 例年10月試験・申込7月頃。要至急確認(締切間近の可能性)

## 優先度: 中
- [ ] 衛生管理者(第一種・第二種) — 各安全衛生技術センターごとに毎月実施
- [ ] 登録販売者 — 都道府県ブロックごとに8〜12月実施
- [ ] ケアマネジャー(介護支援専門員) — 例年10月試験・申込は都道府県ごと6月頃
- [ ] G検定・E資格 — IT系で成長中
- [ ] 秘書検定 — 6月・11月・2月実施
- [ ] カラーコーディネーター/色彩検定
- [ ] 簿記 第175回(2027年2月・2級3級のみ) — 174回終了後に追加

## 追加時のルール
1. 必ず公式(実施団体)サイトで日程を確認し、sources の先頭に公式URLを置く
2. 受験料は公式で確認できた場合のみ記載(不明なら null)
3. description は事実のみ。合格率などの数字は公式発表があるものだけ書く
