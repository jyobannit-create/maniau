# マネタイズ手順

## 収益モデル

「締切を調べに来た人=受験意欲が最高潮の人」に、資格別ページで対策講座を1つだけ紹介する。
CTAは1ページ1つの原則(affiliate.url 未設定時は公式サイトリンクが自動でCTAになる)。

## 申請フォーム記入テンプレ(コピペ用)

| 項目 | 記入内容 |
|------|---------|
| サイト名 | マニアウ |
| サイトURL | https://maniau-shikaku.com/ |
| サイトカテゴリ | 資格・スクール / 学び・教育 |
| 月間PV(見込み) | 〜3,000(開設直後のため見込み) |
| 更新頻度 | 週1回以上(自動更新) |

**サイト紹介文(そのまま使える):**

> 資格試験の「申込締切」に特化した情報サイトです。宅建・行政書士・簿記・TOEIC・FP・情報処理技術者試験など主要16資格の申込期間・試験日を、実施団体の公式発表をもとに週1回更新し、締切が近い順に一覧化しています。各資格ページには情報源と最終確認日を明記し、受験を検討しているユーザーが申込機会を逃さないことを目的としています。今後は掲載資格を順次拡大し、資格対策講座の紹介を行う予定です。

## ステップ1: ASP登録(サイト公開後すぐ)

審査にはサイトが公開済みであることが必要。独自ドメイン推奨。

| ASP | 主な提携先候補 | 備考 |
|-----|---------------|------|
| A8.net | アガルート、ユーキャン、フォーサイト | 資格系の案件数最多。最初に登録 |
| もしもアフィリエイト | スタディング(購入額の約10%) | W報酬制度あり |
| afb | ユーキャン系、通信講座各種 | 資格系の承認率が高いとされる |
| バリューコマース | ヒューマンアカデミー等 | 補助的に |

- 資料請求系: 1件500〜1,000円前後 / 講座購入系: 数千円〜1万円超(講座単価による)
- 各プログラムの規約(リンク改変禁止・表記義務)を確認してから設置する

## ステップ2: 提携承認後の設置

該当資格の JSON を編集するだけ:

```json
"affiliate": {
  "label": "スタディングの宅建講座を見る(無料体験あり)",
  "url": "https://...(ASP発行のリンク)",
  "note": "もしも経由・2026-08承認"
}
```

ビルドすると自動で `rel="sponsored nofollow"` 付きCTAに切り替わる。

## ステップ3: 表記義務(必須)

- アフィリエイトリンクを1つでも設置したら、サイトに「当サイトは広告(アフィリエイトプログラム)を利用しています」の表記を追加する(景表法のステマ規制対応)
- 追加場所: build.mjs のフッター文言。設置時に忘れずに更新すること

## 優先順位(単価×検索量)

1. 宅建 — 受験者数最大・講座案件も最多
2. 簿記2・3級 — 検索量が大きい
3. FP2・3級 — CBT通年化で「いつ受けられる?」検索が増加見込み
4. 行政書士・マン管・賃管 — 講座単価が高い
5. TOEIC — 対策アプリ系(スタディサプリENGLISH等)
